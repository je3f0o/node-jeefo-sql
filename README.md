# jeefo-sqlite3-orm

Simple sqlite3 orm library used for my own private projects.

## Getting Started
Install the module with: `npm install jeefo-sqlite3-orm`

```javascript
var jeefo_sqlite3_orm = require('jeefo-sqlite3-orm');
jeefo_sqlite3_orm.awesome(); // "awesome"
```
